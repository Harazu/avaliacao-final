Comandos Básicos do Git

Configuração Inicial

Configura o nome do usuário e o gmail para as suas contribuições:

    
$ git config --global user.name "Seu Nome"
$ git config --global user.email "seu-email@example.com"

Criação e Clonagem de Repositórios

Inicializa um novo repositório Git:

    
$ git init

Clona um repositório existente para o seu computador:

    
$ git clone URL_do_repositório

Trabalhando com Alterações

Verifica o status atual do seu repositório:

    
$ git status

Adiciona um arquivo para ser incluído no próximo commit:

    
$ git add nome_do_arquivo

Realiza um commit com as alterações adicionadas:

    
$ git commit -m "Mensagem do commit"

Visualização do Histórico

Veja o histórico completo dos commits no repositório:

    
$ git log

Veja uma versão resumida do histórico dos commits:

    
$ git log --oneline

Ramificações (Branches)

Lista todas as ramificações no repositório:

    
$ git branch

Cria uma nova ramificação:

    
$ git branch nome_da_branch

Altera para uma ramificação existente:

    
$ git checkout nome_da_branch

Sincronizando com Repositório Remoto

Adiciona um repositório remoto ao seu repositório local:

    
$ git remote add origin URL_do_repositório

Envia as alterações locais para o repositório remoto:

    
$ git push -u origin nome_da_branch

Atualização do Repositório Local

Atualiza seu repositório local com as alterações mais recentes do repositório remoto:

    
$ git pull origin nome_da_branch
